예어 기술
break 스위치 또는 루프를 종료합니다.
continue 루프에서 점프하여 맨 위에서 시작
debugger JavaScript 실행을 중지하고 디버깅 기능을 호출합니다 (사용 가능한 경우).
do ... while 조건이 참인 동안 문 블록을 실행하고 블록을 반복합니다.
for 조건이 참인 한 실행할 명령문 블록을 표시합니다.
function 함수 선언
if ... else 조건에 따라 실행할 명령문 블록을 표시합니다.
return 함수를 종료합니다.
switch 다른 경우에 따라 실행할 명령문 블록을 표시합니다.
try ... catch 문 블록에 오류 처리를 구현합니다.
var 변수 선언
