[초급이론] HTML 필수이론 정리
구조만들기
  html 문서의 구조 : 태그로 이루어졌으며 <html> 태그 안에 <head>와 <body>로 구분된다.
  head : meta 데이터 / link / style / script등을 적용시킨다.
  body : 실제 브라우저상 노출되는 부분을 작성한다.

텍스트서식
  텍스트를 나누고 강조하고 색을 바꾸는 등의 작업이 가능하다
  <b> : 텍스트를 굵게
  <p> : 문단구분 태그
  <br>: 줄바꿈 태그
  <mark>: 형광처리 등

목록만들기
  순서가 있는 리스트 / 순서가 없는 리스트를 만들수 있다.
  ol 순서가 있는 리스트 : type으로 1, 로마자 등으로 순서를 지정할수 있다.
  ul 순서가 없는 리스트 : type으로 동그라미 네모 등으로 마크를 변경할수 있다.
  리스트안의 리스를 만들때 참고할점
  <ul>
    <li> </li>
      <ul>
	  <li> </li.
	  <li> </li>
      </ul>
    <li> </li>
  </ul>
  이런경우는 리스트 안의 리스트가 부모자식이 아닌 형제의 개념으로 취급되므로
  리스트 안의 리스트는 모 리스트의 목록이 자 리스트를 포함하는 방식으로 작성되어야 한다.
  <ul>
    <li>
      <ul>
	  <li> </li.
	  <li> </li>
      </ul>
    </li>
    <li> </li>
  </ul>

이미지 다루기
  img 태그의 사용방법
  이미지와 글을 함께 적을때 figure 태그를 사용하여 작성 할 수 있다.

하이퍼링크와 앵커 만들기
  하이퍼링크 a 태그를 이용하여 생성할수있다.
  네비게이션창을 만들고 a태그를 이용하여 한 화면안의 특정 지점으로 이동이 가능한데
  a태그 href에 목표로 하는 지점의 아이디 혹은 클래스를 입력해 두면 이동이 가능하다.

이미지맵
  이미지안의 특정 지점에만 링크등을 걸어줄수 있다.
  img
  map
    area
  map
  형태로 작성하면 되고, img 태그의 usemap 속성과 map의 이름을 일치시키고
  area 태그의 속성별로 사용법이 달라진다.
  area shape이 rect 일 경우 좌상단 xy값 우하단 xy값을 순서대로 coords에 입력하면된다.
       circle일 경우 원 중심의 xy값과 반지름을 순서대로 coords에 입력하면 된다.

멀티미디어 다루기
  video / audio / iframe 태그를 이용해 동영상과 음원을 보여줄수있다.
  몇가지 속성으로 autoplay, muted, loop, controls, poster등이 있고
  poster의 경우는 재생이 불가할 경우 대체하는 이미지를 보여준다.

표만들기
  tr : 테이블로우(행)
  td : 테이블데이터(열)
  행을 합칠때(위 아래) rowspan / 열을 합칠때(좌 우) colspan
