CSS의 기본 구조

위의 이미지가 기본적인 CSS ruleset의 형태입니다.

선택자(Selector)
선 언(Declaration)
속 성(Property)
속성값(property value)으로 구성되어 있습니다.

ruleset은 선택자 별로 구분되며 각 ruleset은 반드시 {}로 감싸져야 합니다.
속성과 속성값은 :(콜론)으로 구분되며 여러 속성을 입력할 경우 ;(세미콜론)으로 구분할수 있습니다.
선택자를 하나이상 사용할수 있고 여러가지 선택자를 사용할 경우엔 ,로 구분할수 있습니다.

선택자의 종류

선택자 예시 설명
요소 선택자 p, body, h1 등 이미 지정되어 있는 요소를 선택한다.
아이디 선택자 #id1, #id2 등 사용자가 지정한 아이디를 선택한다.
클래스 선택자 .class1, .class2 등 사용자가 지정한 클래스를 선택합니다.
속성 선택자 img[src] 등 선행한 요소 등의 속성을 선택합니다. 선행한 요소등은 선택하지 않습니다.
수도(Pseudo) 클래스 선택자 a:hover 등의 특정한 요소등의 특정한 행위를 선택합니다. a 태그의 마우스가 위치하면 동작하는 선택자입니다.

박스모델

CSS 레이아웃의 기반은 박스 모델입니다.

1
2
3

파란색 배경의 박스안에 1, 2, 3번이 순서대로 정렬되어 있습니다.
위의 예시처럼 층층히 쌓이는 모습이 박스처럼 보입니다.

박스의 속성으로 3가지를 볼수 있습니다.
padding : 내용/글자/그림등 실제하는 부분 주위를 감싸고 있는 영역입니다.
border : padding의 끝 이자 margin의 시작점 테두리 입니다.
margin : border 바깥의 영역입니다.
